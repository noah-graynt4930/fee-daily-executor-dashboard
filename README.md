## What is FEE Dashboard?

FEE Dashboard provides a web-first view for day-to-day monitoring of executors connected to FEE operations. Packaged as a straightforward HTML dashboard, it gathers reporting-focused information into a format that is easy to open in a browser, inspect, and distribute.

This project is intended for teams that want a dependable daily reference for partner status and executor oversight. Its layout emphasizes monitoring and reporting clarity while keeping access simple through a standard web-based workflow.
